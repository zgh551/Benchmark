# sdnn GUI
![TOP](https://s2.loli.net/2024/08/08/8rJ6j7ydo1FquYm.jpg)
目前包含以下功能：
- 性能数据展示，包括性能数据的波形图展示，profiler数据展示和graph图结构展示；
- 模型精度数据展示，包括分类模型，检测模型和语义分割模型；
- 模型精度分析展示页面，包括相似度，MSE等数据展示；
- 模型注册界面，包括精度评估模型，测试需要的模型和客户的模型；
